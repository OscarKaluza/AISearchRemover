# AISearchRemover

This project is made to remove the AI overview from google.

This is a browser extension that works on all chromium based browsers.

It works by appending `-ai` to your search query. For example:

| Input                     | Output                       |
|---------------------------|------------------------------|
| Never gonna give you up   | Never gonna give you up -ai  |


## Installation

#1. Download this repository
```
git clone https://github.com/Oscarkaluza/AISearchRemover

or click "Download ZIP" above.
```

#2. In your browser
```
1. Go to chrome://extensions/
2. Enable "Developer mode" in the upper right corner
3. In the upper left corner, click on "Load Unpacked"
4. Select the folder "AISearchRemover" that you downloaded.
```

#3. Enjoy AI Free searches
